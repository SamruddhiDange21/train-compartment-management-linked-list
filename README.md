# train-compartment-management-linked-list
A C++ program that simulates a train compartment management system using a singly linked list, supporting insertion, deletion, reversal, and traversal operations.

# Train Compartment Management System

## Description
This project simulates a train compartment management system using a singly linked list in C++.

Each compartment is represented as a node in the linked list, allowing dynamic insertion and deletion of compartments.

## Features
- Add compartment at beginning
- Add compartment at end
- Add compartment at specific position
- Remove compartment
- Display compartments
- Reverse compartment order

## Data Structure Used
Singly Linked List

## Operations Complexity
Insert at beginning: O(1)  
Insert at end: O(n)  
Delete operations: O(n)

## How to Run

Compile the program:

```
g++ train.cpp -o train
```

Run the program:

```
./train  
```
